# utils

## API

- **`ncd`** (Function) — Calculate Normalized Compression Distance between two texts.
- **`compress`** (Function) — Compress text using LZ77-style algorithm.
- **`find_longest_match`** (Function) — Find the longest match in the search window for LZ77 compression.
- **`calculate_adjacent_ncd`** (Function) — Calculate NCD values for sliding window of adjacent texts.
- **`min_max_normalize`** (Function) — Normalize an array of values using Min-Max normalization.
- **`is_trailing_punctuation`** (Function) — Check if a character is trailing punctuation that follows a terminator.
- **`find_next_boundary`** (Function) — Find the next sentence boundary from a given position.
- **`update_quote_stack`** (Function) — Update the quote stack state for one character.
- **`is_at_boundary`** (Function) — Check if a position is at a sentence boundary.
- **`combine_sentences`** (Function) — Combine multiple sentences into a single text string.
- **`find_sentence_boundaries`** (Function) — Find sentence boundary positions in text.
- **`get_closing_quote`** (Function) — Get the matching closing quote for an opening quote.
- **`is_sentence_terminator`** (Function) — Check if a character is a sentence terminator.
- **`get_opening_quote`** (Function) — Get the matching opening quote for a closing quote.
- **`default_quote_safe_max_length`** (Variable) — Default maximum length protected from punctuation splitting inside Japanese quotes.

And 59 more symbols.
