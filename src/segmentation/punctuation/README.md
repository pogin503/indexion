# punctuation

## API

- **`segment_by_punctuation`** (Function) — Segment text using punctuation-based chunking.
- **`PunctuationStrategy`** (Struct) — Punctuation-based segmentation strategy.
- **`segment`** (Function) — Segment text using punctuation strategy with default configuration.
- **`with_config`** (Function) — Create a PunctuationStrategy with custom configuration.
- **`extract_substring`** (Function) — Extract substring from text given character offsets.
- **`new`** (Function) — Create a new PunctuationConfig with default values.
- **`create`** (Function) — Create a PunctuationConfig with custom values.
- **`PunctuationConfig`** (Struct) — Configuration for punctuation-based chunking.
- **`is_whitespace_only`** (Function) — Check if a string contains only whitespace.
- **`default_min_chunk_size`** (Variable) — Default minimum chunk size.
- **`default_max_chunk_size`** (Variable) — Default maximum chunk size.
- **`get_min_chunk_size`** (Function) — Get the minimum chunk size.
- **`get_max_chunk_size`** (Function) — Get the maximum chunk size.
- **`default_target_chunk_size`** (Variable) — Default target chunk size.
- **`get_target_chunk_size`** (Function) — Get the target chunk size.
