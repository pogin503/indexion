# types

## API

- **`ComponentDoc`** (Struct) — Documentation extracted from source file comments.
- **`TokenInfo`** (Struct) — Information about a design token (CSS variable).
- **`new`** (Function) — Create a new ComponentDoc with required name.
- **`make`** (Function) — Create a ComponentDoc with all fields.
- **`with_default`** (Function) — Create a TokenInfo with default value.
- **`with_tokens`** (Function) — Create a TokenGroup with tokens.
- **`TokenGroup`** (Struct) — A group of related tokens.
- **`add_token`** (Function) — Add a token to the group.
- **`get_default_value`** (Function) — Get the default value.
- **`get_css_var`** (Function) — Get the CSS variable.
- **`get_description`** (Function) — Get the description.
