# diagram

## API

- **`generate_dep_diagram`** (Function) — Generate a module dependency diagram in Mermaid format.
- **`normalize_module_label`** (Function) — Normalize module label for display.
- **`format_external_package`** (Function) — Format external package path for display.
- **`format_internal_package`** (Function) — Format internal package path for display.
- **`strip_file_suffix`** (Function) — Strip file suffix from path (e.g., src/kgf/types/moon.pkg -> src/kgf/types).
- **`generate_overview_diagram`** (Function) — Generate a combined overview diagram showing modules and their key symbols.
- **`generate_hierarchy_diagram`** (Function) — Generate a type hierarchy diagram in Mermaid classDiagram format.
- **`find_common_prefix`** (Function) — Find the longest common prefix among all path segment arrays.
- **`generate_dep_diagram_impl`** (Function) — Internal implementation for diagram generation.
- **`strip_prefix_n`** (Function) — Strip n characters from the start of a string.
- **`render_text`** (Function) — Render GraphJSON to plain text adjacency list.
- **`sanitize_class_name`** (Function) — Sanitize class name for Mermaid classDiagram.
- **`render_dot`** (Function) — Render GraphJSON to DOT (Graphviz) format.
- **`escape_mermaid`** (Function) — Escape special characters for Mermaid.
- **`join_lines`** (Function) — Join array of strings with newline.
