# index

## API

- **`HubPage`** (Struct) — A hub page — most-linked page in the wiki (Graphify's "God Nodes").
- **`to_markdown`** (Function) — Render the index as markdown content suitable for a wiki page.
- **`IndexCategory`** (Struct) — A category grouping pages by their source directory.
- **`extract_top_directory`** (Function) — Extract the top-level directory from a file path.
- **`WikiIndex`** (Struct) — The complete index content ready for rendering.
