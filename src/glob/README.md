# glob

## API

- **`expand_glob`** (Function) — Expand a glob pattern against the filesystem and return matching paths.
- **`glob_match_anywhere`** (Function) — Match pattern anywhere in the path (useful for unanchored patterns).
- **`has_glob_pattern`** (Function) — Check whether a string contains glob metacharacters (*, ?, [).
