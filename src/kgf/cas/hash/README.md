# hash

## API

- **`as_content_hash`** (Function) — Create a ContentHash from a string (with validation).
- **`compute_spec_hash`** (Function) — Compute hash of a KGF spec for cache invalidation.
- **`is_valid_hash`** (Function) — Validate that a string looks like a valid ContentHash (32 hex characters).
- **`compute_hash`** (Function) — Compute SHA-256 hash of content, truncated to 32 hex characters.
- **`rotr`** (Function) — Right rotate a 32-bit unsigned integer.
- **`sha256`** (Function) — Pure MoonBit SHA-256 implementation.
- **`string_to_bytes`** (Function) — Convert string to UTF-8 bytes.
- **`sha256_block`** (Function) — SHA-256 compression function.
