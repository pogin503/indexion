# types

## API

- **`AttrAction`** (Struct) — AttrAction represents an attribute action in the grammar.
- **`SemOnBlock`** (Struct) — SemOnBlock represents a semantic on-block definition.
- **`get_start_rule`** (Function) — Returns the start rule name.
- **`get_value`** (Function) — Returns the value of the token (first capture group when present).
- **`PreprocessFstSpec`** (Struct) — PreprocessFstSpec represents a JSON-defined preprocess FST specification.
- **`ResolutionStats`** (Struct) — ResolutionStats tracks statistics for module resolution performance.
- **`is_resolving`** (Function) — Checks if a module is currently being resolved (circular detection).
- **`ResolutionCache`** (Struct) — ResolutionCache caches module resolution results for performance.
- **`ResolverSpec`** (Struct) — ResolverSpec represents the configuration for module resolution.
- **`get_strip`** (Function) — Returns the number of characters to strip from start for value.
- **`FstTransition`** (Struct) — FstTransition represents a single state transition in the FST.
- **`ResolveStep`** (Struct) — ResolveStep represents a single step in the resolution chain.
- **`ResolverAlias`** (Struct) — ResolverAlias represents a path alias for module resolution.
- **`get_project_config`** (Function) — Returns the project config file name (e.g., moon.mod.json).
- **`Tok`** (Struct) — Tok represents a lexed token with position information.
