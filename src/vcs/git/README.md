# git

## API

- **`last_commit_seconds`** (Function) — Return the last commit timestamp in seconds for the given path, or None on failure.
- **`status_dirty`** (Function) — Return whether the given path has uncommitted changes in the Git working tree.
- **`file_mtime_seconds`** (Function) — Return the file modification time in seconds, or None if unavailable.
- **`query_path`** (Function) — Compute the relative path from the repository root for Git queries.
