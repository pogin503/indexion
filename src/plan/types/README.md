# types

## API

- **`with_priority`** (Function) — Create an unchecked checklist item with the given priority.
- **`ConvertContext`** (Struct) — Conversion context for building PlanDocument.
- **`SectionContent`** (Enum) — Content types that can appear in a section.
- **`ChecklistItem`** (Struct) — A checklist item with priority and status.
- **`simple`** (Function) — Create a list item with no description.
- **`ListItem`** (Struct) — A list item with optional description.
- **`with_desc`** (Function) — Create a list item with a description.
- **`new`** (Function) — Create a new empty plan document.
- **`PlanSection`** (Struct) — A section within a plan document.
- **`TableData`** (Struct) — Table data with headers and rows.
- **`nested`** (Function) — Create nested sections content.
- **`add_section`** (Function) — Add a section to the document.
- **`default_left`** (Function) — Get default alignment (Left).
- **`to_json`** (Function) — Convert PlanDocument to JSON.
- **`add_row`** (Function) — Append a row to the table.
