# tokenizer

## API

- **`tokenize_for_tfidf`** (Function) — Tokenize text for TF-IDF vectorization.
- **`extract_character_bigrams`** (Function) — Extract character bigrams from text for CJK processing.
- **`normalize_text`** (Function) — Normalize text for TF-IDF processing.
- **`extract_word_tokens`** (Function) — Extract word tokens from text.
- **`is_cjk_char`** (Function) — Check if a character is a CJK (Chinese/Japanese/Korean) character.
- **`is_word_char`** (Function) — Check if a character is a letter or digit (for word tokenization).
- **`to_lower`** (Function) — Convert a character to lowercase (ASCII only).
