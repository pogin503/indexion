# extract

## API

- **`default`** (Function) — Create default extraction config.
- **`should_include_kind`** (Function) — Check if a symbol kind should be included (not in exclude list).
- **`all`** (Function) — Create config that includes all symbols (no exclusions).
- **`get_max_depth`** (Function) — Get the maximum depth in the call graph.
- **`ExtractConfig`** (Struct) — Configuration for function extraction.
