# scope

## API

- **`new`** (Function) — Create a PathSubject from a primary path and optional alternates.
- **`is_empty`** (Function) — Return true if the scope has no include or exclude patterns.
- **`PathScope`** (Struct) — Common path scope matching for command-level selectors.
- **`subject_matches_pattern`** (Function) — Check whether a subject matches a single scope pattern.
- **`matches`** (Function) — Check whether a path subject is included by the scope.
- **`matches_path`** (Function) — Check whether a single path is included by the scope.
- **`path_matches_pattern`** (Function) — Match a single path against a single glob pattern.
