# apted

## API

- **`calculate_tsed`** (Function) — Calculate TSED similarity between two trees.
- **`with_finalized_labels`** (Function) — Create a new tree with finalized labels.
- **`should_compare_trees`** (Function) — Early pruning: determine if two trees are worth comparing.
- **`compute_tree_edit_distance`** (Function) — Compute tree edit distance between two trees using Zhang-Shasha algorithm.
- **`compute_forest_dist`** (Function) — Compute forest distances for a pair of keyroots.
- **`IndexedTree`** (Struct) — Indexed node information for Zhang-Shasha algorithm.
- **`calculate_tsed_distance`** (Function) — Calculate TSED distance between two trees.
- **`peg_cost_config`** (Function) — Cost configuration optimized for PEG AST comparison.
- **`tree_similarity`** (Function) — Compute tree similarity (1 - normalized distance).
- **`normalized_tree_edit_distance`** (Function) — Compute normalized tree edit distance.
- **`NodeInfo`** (Struct) — Node information for APTED algorithm indexing.
- **`fill_postorder`** (Function) — Fill arrays using postorder traversal.
- **`TreeNode`** (Struct) — Generic tree node representation for tree edit distance computation.
- **`TSEDConfig`** (Struct) — Configuration for TSED (Tree Structure Edit Distance) calculation.
- **`size`** (Function) — Calculate the size of the tree (total number of nodes).
