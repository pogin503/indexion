# similarity

## API

- **`calculate`** (Function) — Calculate similarity with automatic dispatch based on input type.
- **`parse_bracket_tree`** (Function) — Parse bracket notation into TreeNode.
- **`text_similarity`** (Function) — Calculate text similarity using the strategy.
- **`is_text_strategy`** (Function) — Check if strategy operates on text (vs. trees).
- **`parse_with_weights`** (Function) — Parse strategy with optional hybrid weights.
- **`parse_tree_recursive`** (Function) — Recursive tree parser for bracket notation.
- **`StrategyKind`** (Enum) — Strategy kind enum for unified handling.
- **`parse`** (Function) — Parse strategy name to StrategyKind.
- **`is_tree_strategy`** (Function) — Check if strategy operates on trees.
- **`available_strategies`** (Function) — List all available strategy names.
- **`text_strategies`** (Function) — List text-only strategy names.
- **`tree_strategies`** (Function) — List tree-only strategy names.
