# tfidf_strategy

## API

- **`all_pairs_above_threshold`** (Function) — Calculate all pairwise similarities above a threshold.
- **`TfidfStrategy`** (Struct) — TF-IDF based similarity strategy.
- **`TfidfBatch`** (Struct) — Batch TF-IDF similarity calculator.
- **`from_tokens`** (Function) — Create a batch calculator from pre-tokenized documents.
- **`similarity`** (Function) — Calculate similarity between two documents by index.
- **`length`** (Function) — Get the number of documents in the batch.
- **`new`** (Function) — Create a new TfidfStrategy.
