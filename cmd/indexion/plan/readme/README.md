## API

- **`get_plan_filename`** (Function) — Convert include path to plan filename.
- **`generate_writing_plan`** (Function) — Generate writing plan content for a specific include path.
- **`extract_include_paths`** (Function) — Extract all {{include:...}} paths from template content.
- **`get_section_guidance`** (Function) — Get section-specific writing guidance based on filename.
- **`extract_doc_title`** (Function) — Extract title from first heading in markdown file.
- **`command`** (Function) — Define the readme plan command for argparse.
- **`ReadmePlanConfig`** (Struct) — Configuration for plan readme command.
- **`render_plans_markdown`** (Function) — Render summary in markdown format.
- **`ExistingDoc`** (Struct) — Existing documentation file info.
- **`default`** (Function) — Create default ReadmePlanConfig.
- **`render_plans_json`** (Function) — Render summary in JSON format.
- **`create_dir_recursive`** (Function) — Create directory recursively.
