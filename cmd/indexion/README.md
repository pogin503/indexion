# indexion

Source code exploration and documentation tool.

## Usage

```bash
indexion <command> [options]
```

## Commands

| Command | Description |
|---------|-------------|
| `init` | Initialize indexion configuration |
| `explore` | Analyze similarity across files in a directory |
| `plan` | Generate planning documents (refactor, solid, unwrap, etc.) |
| `doc` | Documentation commands (graph, readme, init) |
| `kgf` | KGF spec management and inspection |
| `digest` | Purpose-based function index |
| `sim` | Calculate text similarity/distance between two texts |
| `segment` | Split text into contextual segments |
| `perf` | Performance benchmark commands |
| `update` | Check for updates and install the latest version |
| `serve` | Start HTTP server for codebase search |
| `grep` | KGF-aware token pattern search |
| `search` | Semantic search across code, wiki, and documentation |
| `mcp` | Start MCP server exposing indexion tools |
| `spec` | Specification-driven analysis (verify conformance, etc.) |
| `wiki` | Wiki management (pages, index, lint, export, import, log, hook) |

## Global Options

| Option | Description |
|--------|-------------|
| `-h, --help` | Show help information |
| `-V, --version` | Show version information |
